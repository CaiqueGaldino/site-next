# Componentes Mobile

Este diretório contém os componentes otimizados para dispositivos móveis.

## Características dos componentes mobile:

- Layout adaptado para telas pequenas
- Otimização de performance para redes móveis
- Interações touch-friendly
- Feedback háptico quando disponível
- Carregamento lazy de imagens
- Animações simplificadas para melhor performance

## Como usar:

Importe os componentes desta pasta quando estiver desenvolvendo a versão mobile do site.

```tsx
import HeaderMobile from "@/components/mobile/HeaderMobile";
```
