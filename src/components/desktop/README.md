# Componentes Desktop

Este diretório contém os componentes otimizados para dispositivos desktop.

## Características dos componentes desktop:

- Layout expansivo aproveitando telas maiores
- Hover effects e interações com mouse
- Animações mais elaboradas
- Recursos visuais aprimorados
- Suporte a múltiplas colunas
- Melhor uso de espaço horizontal

## Como usar:

Importe os componentes desta pasta quando estiver desenvolvendo a versão desktop do site.

```tsx
import HeaderDesktop from "@/components/desktop/HeaderDesktop";
```
